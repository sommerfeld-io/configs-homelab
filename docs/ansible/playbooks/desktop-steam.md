# Ansible Playbook - Desktop Steam

This Ansible playbook installs and prepares [Steam](https://store.steampowered.com) gaming platform on desktop workstations. The playbook handles the initial Steam installation and provides post-installation guidance to ensure optimal gaming configuration.

## What it does

- **Steam Installation**: Installs the Steam gaming client on desktop systems
- **Post-Installation Guidance**: Provides instructions for completing the Steam configuration including account login, beta client activation, and Proton compatibility layer setup

The playbook targets desktop workstations.
